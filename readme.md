# AI学习项目

这是一个使用Python和LangChain框架开发的AI学习项目。

## 功能特点

- 基础LLM调用示例
- 聊天模型示例
- 使用工具的Agent示例
- 文档问答功能

## 环境要求

- Python 3.8+
- OpenAI API密钥
- 其他依赖包（见requirements.txt）

## 安装

1. 克隆仓库：
```bash
git clone [你的仓库地址]
```

2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 配置环境变量：
创建.env文件并添加你的OpenAI API密钥：
```
OPENAI_API_KEY=你的API密钥
```

## 使用方法

运行主程序：
```bash
python test.py
```
